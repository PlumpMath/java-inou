INOU

  Integrated Numerical Operation Utilities.

  Version 0.1.0
  Time-stamp: "2011-08-11 22:20:18 sakurai"

■何か？

Pure Java による可視化（グラフ、3D）、数値計算ライブラリ。

■使用例

samples/
  graph/D2Demo : 2次元グラフ
  graph/D3Demo : 3次元グラフ
  d3/Demo      : 3D デモ
  d3/Sample    : 3D サンプル

■インストールと設定

inou.jar をクラスパスに加える。
